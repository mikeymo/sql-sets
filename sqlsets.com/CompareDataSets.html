Compare DataSets With Ease

Often times in the course of writing or testing software it is convenient to be able to compare datasets (i.e. compare query results) to see what has changed over some period of time or series of unit tests. With SQL Sets you can easily compare data sets and view differences in a simple and intuitive way.

So, in effect, you can take the query results that were saved to a Portable Data Set Document file a week ago and compare it to the same query results run today. This compare set can then be annotated, bookmarked and shared like any other Portable Data Set Document.

This provides a powerful mechanism for validating software changes, data manipulation tasks and general collaboration. Comparing datasets has never been easier! Check out other SQL Set Features or other SQL Set Uses.
